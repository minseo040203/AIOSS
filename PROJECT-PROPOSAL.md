## 기본 정보 (Basic Info)

### 프로젝트명
**라즈베리파이 기반 재난안전 정보 RAG AI 비서 개발**
(Raspberry Pi-based Disaster Safety Information RAG AI Assistant Development)

### 개요
오프라인 환경에서도 신속하게 재난안전 정보에 접근할 수 있는 **Local LLM + RAG 기반 AI 비서 시스템**을 라즈베리파이에 구현합니다.

기존의 클라우드 기반 AI 서비스는 네트워크 의존성이 높고 개인정보 유출 위험이 있습니다. 본 프로젝트는 완전히 오프라인에서 동작하면서도 재난 상황에서 신속한 정보 제공이 가능한 시스템을 개발합니다.

### 동기 (Why This Project)
**Why we started this project?**
- 재난 상황에서 인터넷 연결이 끊어져도 즉시 대응할 수 있는 독립적인 시스템 필요
- 개인정보 보호와 오프라인 운영이 가능한 로컬 AI 시스템의 필요성 증대
- 저사양 임베디드 기기(라즈베리파이)에서 LLM + RAG을 효율적으로 운영하는 기술 확보

### 예상 결과물
- **완성된 시스템**: 라즈베리파이상에서 구동되는 오프라인 RAG AI 비서 완제품
- **웹 UI**: Streamlit/FastAPI 기반 직관적 사용자 인터페이스
- **재난안전 DB**: 재난 매뉴얼 및 안전 가이드 벡터 데이터베이스
- **성능 분석 보고서**: 임베디드 환경에서의 LLM 최적화 성과

## 기술 스택 (Tech Stack)

### Frontend
- **Streamlit** - 웹 기반 사용자 인터페이스
- **FastAPI** - RESTful API 서버
- HTML/CSS/JavaScript - UI 커스터마이징

### Backend
- **Python 3.8+** - 주 개발 언어
- **Ollama** - Local LLM 인퍼런스 엔진
- **Llama 2 / Mistral** - 경량 언어 모델
- **LangChain** - LLM 애플리케이션 프레임워크

### Database
- **ChromaDB** - 벡터 데이터베이스 (권장)
- **FAISS** - Meta의 벡터 검색 라이브러리 (대안)
- **JSON** - 경량 문서 저장소

### Deployment & Infrastructure
- **Raspberry Pi OS** - 라즈베리파이 운영체제
- **Linux/Ubuntu** - 개발 및 배포 환경
- **Git/GitHub** - 버전 관리
- **Docker** (선택사항) - 컨테이너화

## 주요 기능 (Key Features)

### 1️⃣ 오프라인 LLM 기반 질의응답 (Core Feature)
- 인터넷 연결 없이 완전히 독립적으로 동작
- 라즈베리파이 환경에 최적화된 경량 LLM 실행
- 5초 이내 빠른 응답 속도 보장

### 2️⃣ RAG 기반 재난안전 정보 검색 및 제공 (Core Feature)
- 공개 재난/안전 매뉴얼(PDF) 자동 수집 및 처리
- 자연어 기반 의미론적 검색
- 출처 명시 및 원본 문서 인용
- 관련 문서 자동 추출 및 답변 생성

### 3️⃣ 임베디드 환경 최적화 및 성능 분석 (Core Feature)
- 모델 양자화 (4bit/8bit) - 메모리 효율성
- 응답 캐싱 기반 질의 가속화
- CPU/RAM/저장공간 사용량 모니터링
- 성능 벤치마크 및 최적화 보고서

### 부가 기능
- 문서 업로드 및 동적 업데이트
- 신뢰도 지수 표시
- 다국어 지원 (한영)
- 음성 입출력 지원 (선택사항)
